**atspad** is an online playground for programming in [ATS Programming Language](http://www.ats-lang.org). You can write and run your ATS code on full featured linux containers, and share it with the rest of the world *(comming soon)*.

# Features

Although **atspad** is still in beta, it packs a lot of cool features that I would like to present here.

### Editor

**atspad** is using the best web editor [Ace Editor](http://ace.c9.io/), giving you the desktop-level ability to code in the cloud. Plus, it comes with ATS syntax and tons of other language highlighting.

### Runtime

You have a full machine to run your code, and do anything else! This is proudly backed by [Docker](http://www.docker.com/). Through the terminal, you have root access to a dedicated docker container, where you run your code, execute scripts, or just use it as a virtual machine.
